TCRC Youtso Unicode 1.4.5
=========================

- Removed continuous Excel polling, which could show a busy cursor and
  interfere with rapidly typed numbers.
- Excel formula checks now run only after formula commits, calculation events,
  or relevant mouse actions. Ordinary typing and arrow navigation do not scan
  the worksheet.
- Moving between Excel cells with Tab, Enter, arrow keys, or the mouse now
  keeps TCRC Youtso Unicode active until Tibetan typing is turned off.
- SUM, AVERAGE, AutoSum, and other formula result cells now switch to TCRC
  Youtso Unicode after Excel finishes recalculating active or inactive sheets.
- Formula results separated across different rows and columns are formatted
  individually instead of only formatting the first result area.
- A deferred Excel reconciliation check corrects formula result fonts when an
  Office version omits or delays calculation events.
- The digits 0-9 in the font are now narrow Tibetan numerals with the same
  width as western digits: number columns sized before the change no longer
  overflow into "######".
- While Tibetan typing is ON, every Excel cell receives TCRC Youtso Unicode
  before typing, including after Tab, Enter, arrow-key, or mouse navigation.

User experience
---------------
- One installer provides the font, keyboard, converter, and uninstaller.
- A visible TCRC Document Converter shortcut is installed on the desktop.
- Users choose or drag Word, PowerPoint, or Excel files and click Convert to
  Unicode.
- The converter supports .doc/.docx, .ppt/.pptx, and .xls/.xlsx files.
- The converter creates a new modern-format (Unicode) file and preserves the
  original.
- Office-file right-click commands open the same visible converter.
- Ctrl+Alt+N prepares selected Excel cells for Tibetan-looking numbers.
- Excel stores real numeric values, so SUM and arithmetic continue to work.
- Tibetan mode selects TCRC Youtso Unicode automatically in Word, Excel, and
  PowerPoint.
- Photoshop, Illustrator, and InDesign conversion scripts are included.

Reliability and safety
----------------------
- The keyboard now handles keyboard input only.
- Background Microsoft Word monitoring and COM attachment were removed.
- Number handling is limited to the explicit Microsoft Excel workflow.
- The installer no longer replaces Microsoft Himalaya globally.
- Old font-substitution settings created by earlier releases are removed or
  restored during upgrade.
- Installer staging is recreated on every build to prevent stale files.
- The AutoHotkey runtime download is pinned and checksum-verified.
- Product-specific process names avoid stopping unrelated AutoHotkey scripts.
- Excel conversion follows cell font styles, handles inline strings, and
  preserves shared strings used by both legacy and normal-font cells.
- PowerPoint conversion follows direct and inherited paragraph font settings.

Verification
------------
- Python and PowerShell converters produce matching Word XML.
- Python and PowerShell converters cover PowerPoint inherited fonts and Excel
  inline/mixed shared strings.
- Ordinary accented Latin text is preserved during Excel conversion.
- The Windows test generates a legacy document and verifies its expected
  Unicode Tibetan conversion.
- Windows tests cover installation, shortcuts, registry integration,
  conversion, source preservation, Microsoft Word, and Excel calculation.
